## 开发任务

任务:

完成以上修改任务

目标:
- 具体功能按需求实现
- 测试通过
- 代码质量符合要求

要求:
- 想清楚再行动
- 简单优先
- 外科手术式改动: 只做必要的修改，避免引入新的问题, 每一处改动，都要能直接追溯到我的需求
- 作为项目负责人，协调团队成员,充分发挥多agent并发协作的优势，为最终任务结果负责; 如果需要团队之外的角色帮忙, 通过subagent工具进行任务委派

团队:
- coder : 资深开发人员，负责开发具体功能或修改已知问题
- tester : 独立上下文, 负责测试代码并报告问题
- reviewer : 独立上下文, 负责审核代码质量和功能实现
